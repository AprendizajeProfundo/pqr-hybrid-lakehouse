# apps/simulator/

Esta carpeta contiene el código Python para generar datos sintéticos PQRS.

- `generate_pqrs_data.py`: Script principal para simulación determinista.
- `models/`: Modelos de datos y esquemas.
- `utils/`: Utilidades para Faker, seeds, etc.

Ejecutar: `python generate_pqrs_data.py --config docs/07-config/pqrs_simulation_v1.yaml`
